# Football Manager (Console App)

My first C# project built while learning object-oriented programming.

## Features

- Create your own football club
- Choose starting budget based on difficulty
- Automatically generate a 20-player squad
- Random player names
- Random player levels (1–5)
- Random player prices based on level
- Buy players from the transfer market
- Sell players from your squad
- View squad average level
- Generate AI teams and a league
